# This includes the steps involving the setup of the GIT in the local PC 

1. Create an account in the GIT HUB which will be the remote server
2. In the locaL pc  create ssh public key by 'ssh-keygen'  command . Which will 
   generate  the new RSA public key for the SSH connection .
3. Copy this public Id which will be in the ~/.ssh/id_rsa.pub and add this to your remote
   count ssh kyes. (Settings->ssh)
4. Then clone the repo by ssh address
